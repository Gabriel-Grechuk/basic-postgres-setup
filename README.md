# Basic Prosgres Setup

## Required packages
* `docker`
* `docker compose`

## Usage
### To start the postgres
Just need to run the `run.sh` script.
``` shell
$ ./run.sh
```
or, if it requires permissions
``` shell
$ sudo ./run.sh
```

### Stop the docker instance
``` shell
$ ./stop.sh 
```
or
``` shell
$ sudo ./stop.sh
```

### To remove the docker image.
``` shell
$ ./remove.sh
```
or
``` shell
$ sudo ./remove.sh
```
